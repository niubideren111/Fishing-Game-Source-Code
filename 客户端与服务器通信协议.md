### <a name="0-pane"></a>概要
本文档定义了im客户端与Server之间的通讯协议。数据格式为二进制的字节流，通讯数据包括包头与包体部分，包头部分为固定长度的字节流，包体部分格式为msgpack格式打包的字节流。
包体部分打包解包不是使用键值对的方式，而是使用数组的方式，按顺序把值的每一项打包解包，如果是自定义类型，则在自定义类型字段处展开，顺序打包解包。如果字段是数组格式，则先打包解包数组长度，再按顺序打包解包数组。
### <a name="1-pane"></a>1. 通讯包总体结构
#### <a name="1.1-pane"></a>1.1 客户端到服务端的总体包格式
名称     |  长度（字节） |    描述
--------|--------------|----------------
len  | 4  | 整个包长度
stx  | 1  | start of text，固定0x09
ver  | 2  | 协议版本号，用于协议兼容
zip  | 1  | 压缩标识，直接hardcode，1个字节，0:不压缩，1：gzip（用zlib实现），2：7zip
cmd  | 4  | 命令号
seqno  | 4  | 序列号，递增
ext1  | 4  | 目前存放请求UID
ext2  | 4  | 扩展字段2，4个字节，直接hardcode
body  | 不定长  | 包体，不同的cmd对应到不同的包体,body所有的打包用msgpack格式生成

#### <a name="1.2-pane"></a>1.2 服务端到客户端的总体包格式
名称     |  长度（字节） |    描述
--------|--------------|----------------
len  | 4  | 整个包长度
stx  | 1  | start of text，固定0x09
ver  | 2  | 协议版本号，用于协议兼容
zip  | 1  | 压缩标识，直接hardcode，1个字节，0：不压缩，1：gzip（用zlib实现），2：7zip
cmd  | 4  | 命令号
seqno  | 4  | 序列号，递增
timestamp  | 8  | 服务端当前时间戳，精确到毫秒，8个字节
ext1  | 4  | 扩展字段1，4个字节，直接hardcode
ext2  | 4  | 扩展字段2，4个字节，直接hardcode
body  | 不定长  | 包体，不同的cmd对应到不同的包体,body所有的打包用msgpack格式生成

### <a name="2-pane"></a>2. login业务协议(10)
#### <a name="2.1-pane"></a>2.1 获取游戏服ip端口(10000)
* C2S

	名称     |  类型 |    描述
--------|--------------|----------------
agent   | int | 代理(platform)，区分不同的SDK，以及验证方式,0-游客，1-注册账号，5以上是各个SDK
account_id   | string | 平台用户唯一标识
token  | string    | 签名串

* S2C
	名称     |  类型 |    描述
--------|--------------|----------------
ip | string | 游戏服ip
port | int | 游戏服端口
timestamp | int | 时间戳
token | string | 签名串
code | int | 错误码，0为成功

### <a name="3-pane"></a>3. 排行榜业务协议(20)
#### <a name="3.1-pane"></a>3.1 获取排行榜(20000)
* C2S

	名称     |  类型 |    描述
--------|--------------|----------------
rank_type   | int | 3:金币排行榜 6:等级排行榜 7:弹头排行榜

* S2C

	名称 |  类型                      |    描述
--------|-------------------------- |----------------
list   | vector<proto_rank_data>    | 排行榜数据


* proto_rank_data

	名称     |  类型    |    描述
------------|----------|--------------------
pid         | int      | 玩家ID       
nick        | string   | 名字
facelook    | string   | 头像
vip         | int      | VIP
value       | int      | 值
value2      | vector<proto_rank_value> | 值
sign        | int | 个性签名

* proto_rank_value
	名称     |  类型    |    描述
------------|----------|--------------------
key         | int      | 道具ID
val         | int      | 道具数量
